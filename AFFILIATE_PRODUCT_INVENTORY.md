## Where to Add Products

### For CSV Files
Follow this pattern:

```csv
category,subgroup,title,notes,amazon_url,bucket_key,bucket_label,priority
[category],[sub],[Product Name],[Details],[URL],[key],[Label],[1-10]
```

**Example**:
```csv
tanks,glass,Aqueon Standard Glass 20G Long,20x10x12",https://amzn.to/xxx,g_20,20 Gallon,1
```

### For JSON Files
Follow filter.json pattern:

```json
{
  "id": "brand-model-size",
  "brand": "Brand",
  "name": "Full Product Name",
  "type": "CATEGORY",
  "specification": 100,
  "minGallons": 10,
  "maxGallons": 30,
  "url": "https://amzn.to/shortcode"
}
```

### For HTML (Featured Products)
Add to `gear/index.html` within existing card structure:

```html
<a class="btn btn-amazon"
   href="https://www.amazon.com/dp/ASIN?&linkCode=ll1&tag=fishkeepingli-20">
   View on Amazon
</a>
```
