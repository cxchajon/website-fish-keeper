Cycling Coach Prototype FAQ

Do existing automation scripts still run?

Yes. Consent mode, ad loading, nav injection, and footer fetching scripts mirror production. Only headings and structured data were tuned for Gemini requirements.

Will the prototype affect live indexing?

No. The page is blocked with noindex, nofollow directives, so search engines ignore the staging URL while QA takes place.

Where can I report feedback?

Share notes with the Cycling Coach working group or use the contact form so we can track layout, copy, or data issues before launch.
